# תחום אי-ודאות והסתברות - חטיבת ביניים (כיתות ז'-ח')
### מאגר שאלות משרד החינוך ודפי עבודה מותאמים להדפסה A4

יישום זה פותח במטרה להנגיש בצורה מדוייקת ומקצועית ביותר את מאגר השאלות והמשימות הרשמיות של משרד החינוך בנושא **הסתברות, שכיחות יחסית וניתוח נתונים (סטטיסטיקה)** עבור מורי ותלמידי חטיבת הביניים בישראל.

---

## 📘 דף כללים ונוסחאות מרכזיים (תוכנית הלימודים לחלוקה בכיתה)

### 1. שכיחות ושכיחות יחסית
* **שכיחות מוחלטת (Absolute Frequency):** מספר הפעמים שערך או אירוע מסוים מופיע במדגם או בתצפית.
* **השלם (Total):** סך כל התצפיות או הנתונים שנאספו (N).
* **שכיחות יחסית (Relative Frequency):** היחס בין השכיחות המוחלטת של הערך לבין סך כל התצפיות (השלם).
  $$\text{שכיחות יחסית} = \frac{\text{שכיחות מוחלטת}}{\text{סך כל הנתונים}}$$
  * ניתן להציג שכיחות יחסית כ**שבר פשוט**, **שבר עשרוני** או **אחוז**. סכום כל השכיחויות היחסיות תמיד שווה ל-1 (או 100%).

### 2. מאורעות והסתברות קלאסית (מאורע פשוט)
בכל ניסוי מקרי שבו לכל התוצאות יש סיכוי שווה להתקבל (כמו הטלת קובייה הוגנת או סיבוב גלגל מזל שווה-חלוקה), ההסתברות של מאורע שווה ל:
$$P(A) = \frac{\text{מספר האפשרויות הרצויות (החלקיות)}}{\text{סך כל האפשרויות במרחב המדגם (השלם)}}$$

### 3. מאורע משלים (Complementary Event)
* אירוע משלים הוא האירוע שבו המאורע המקורי **אינו** מתרחש.
* סכום ההסתברויות של מאורע והמאורע המשלים שלו תמיד שווה ל-1.
  $$P(\bar{A}) = 1 - P(A)$$

### 4. חשיבה ביקורתית והטיות סטטיסטיות
* **קטיעת ציר ה-y (Y-Axis Cropping):** הצגת גרף עמודות שבו ציר ה-y אינו מתחיל מ-0 (אלא למשל מ-80% או 6,000 תלמידים) מעוותת את קנה המידה חזותית ומייצרת רושם מטעה של הבדל עצום בין הקטגוריות.
* **ספירת כלי רכב לעומת ספירת אנשים:** בקבלת החלטות ציבוריות (כמו הרחבת נתיב אוטובוסים לעומת נתיב מכוניות), הבנה של שכיחות יחסית של המשתמשים בפועל בכל אמצעי תחבורה (ולא רק כמות הרכבים היבשה) מציגה את התמונה האמתית וההוגנת ביותר.
